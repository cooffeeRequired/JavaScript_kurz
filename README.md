## Příklady:

### Pole:

```javascript
let ovoce = ['jablko', 'banán', 'pomeranč'];
console.log(ovoce[1]); // vypíše 'banán'
```

### Map (Mapování):

```javascript
let object = {}
object['klíč'] = 1;
console.log(object['klíč']); // vypíše '1'
```

### Lambda Funkce (Arrow Funkce):

```javascript
let soucet = (a, b) => a + b;
console.log(soucet(5, 3)); // vypíše 8
```
